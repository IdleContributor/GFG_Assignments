## 🛠️ Tech Stack

### Frontend
- **HTML5 & CSS3** - Semantic markup and modern styling
- **JavaScript (ES6+)** - Vanilla JS and modern features
- **React** - Component-based UI library
- **React Router** - Client-side routing
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library

### Backend
- **Node.js** - JavaScript runtime
- **Express** - Web application framework
- **MongoDB** - NoSQL database
- **Mongoose** - MongoDB object modeling
- **JWT** - JSON Web Tokens for authentication
- **bcrypt** - Password hashing
- **Nodemailer** - Email sending
- **Google OAuth** - Social authentication

### Deployment
- **Vercel** - Frontend hosting (A1-A6, A10 frontend)
- **Railway** - Backend hosting (A7-A10 backends)
- **MongoDB Atlas** - Database hosting

## 🚀 Local Development

Each assignment has its own README with setup instructions. General steps:

### Static Sites (A1-A4)
```bash
cd Assignments/A[1-4]
# Open index.html in browser
```

### React Apps (A5-A6)
```bash
cd Assignments/A[5-6]
npm install
npm run dev
```

### Node.js Apps (A7-A8)
```bash
cd Assignments/A[7-8]
npm install
npm start
```

### MERN Apps (A9-A10)

**Backend:**
```bash
cd Assignments/A[9-10]/backend
npm install
# Add .env file with MongoDB URI
npm start
```

**Frontend (A10 only):**
```bash
cd Assignments/A10/frontend
npm install
# Add .env file with backend URL
npm run dev
```
